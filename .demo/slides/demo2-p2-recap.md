# Recap - Esclation to Agent in Teams

> Student flow (IT Support Portal - Front-end)

1. Create a new chat thread
2. Student Join the thread
3. We broadcast the new chat request to all agents

> Agents flow (Teams App)

1. Get Agent Azure Communication Services identity
2. Available Agent in Teams accept the chat request initiated by the student flow
3. Agent Join the thread
4. Update the chat status
