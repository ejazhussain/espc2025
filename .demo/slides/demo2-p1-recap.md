# Recap - IT Support Agent Flow


> Student Query  
- Student submits a question or issue
- Message is sent to the AI support agent
  
> AI Processing with RAG  
- Agent searches knowledge base for relevant information
- AI analyzes the query context and intent
- Combines retrieved data with AI capabilities to formulate answer

> Response Delivery
- Response is delivered directly back to the student
